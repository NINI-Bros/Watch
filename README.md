# 🕶️ The Watch Study
계획을 세우고, 실행하고, 감시하는 회고 스터디

## ✨회고 주제
- 계획 세우고 액션 (운동, 공부, 코딩 등등)
- 서로의 안부를 확인하며 함께 기운 복돋아주기

## 🚀 스터디 진행 기간
- 시작시간 : 22시 (오후10시)
- 횟수 : 주 6회 (주말 제외)
- 매월 기수모집 (11월 베타 테스트)
  
## 🎉 진행방식
- 불참은 미리 말하고 체크해두기
- 1인 10분 스피치
- 피드백 1인 5분 또는 알아서 DM
- 인원 맞춰 시간 조정
- 사회자는 모든 인원 돌아가면서
- 회고 양식에 맞춰 미리 작성
